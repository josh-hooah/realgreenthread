# GREENTHREAD - Premium African Fashion E-Commerce Website

A full-featured e-commerce website for GREENTHREAD, a clothing brand specializing in premium African menswear including senator wear, kaftan, suits, corporate wear, shoes, and accessories.

## 🎨 Design Theme
- **Primary Color**: Black (#1a1a1a)
- **Accent Color**: Gold (#d4af37)
- **Secondary Color**: White (#ffffff)

## ✨ Features

### 1. **Product Catalog**
- Browse all products by category (Senator Wear, Kaftan, Suit, Corporate Wear, Shoes, Accessories)
- Sort products by name, price (low to high), and price (high to low)
- Filter by category
- Product details with images, descriptions, and pricing
- 15+ pre-made products with detailed information

### 2. **Shopping Cart**
- Add products to cart with selected color, size, and quantity
- View cart items with total price calculation
- Update quantities and remove items
- Persistent cart state during session
- Attractive cart summary

### 3. **Custom Cloth Booking**
- Fill a detailed form to request custom tailored clothes
- Specify cloth type, color preferences, measurements, and delivery date
- Send booking request directly to WhatsApp (+234 905 881 0096)
- Form validation and success confirmation

### 4. **Checkout & Ordering**
- Enter delivery information (name, phone, email, address, date)
- Select payment method (Direct Payment, Bank Transfer, Cash on Delivery)
- Review order before submission
- Send complete order details to WhatsApp
- Order confirmation messages

### 5. **WhatsApp Integration**
- Send booking requests to WhatsApp
- Send cart orders to WhatsApp
- Automatically formatted messages with all details
- One-click WhatsApp link opening
- Contact: +234 905 881 0096

### 6. **Responsive Design**
- Fully responsive on desktop, tablet, and mobile devices
- Smooth animations and transitions
- Professional UI with navigation and footer

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Navigate to the project directory:
```bash
cd greenthread
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open your browser and navigate to:
```
http://localhost:3000
```

## 🔧 Usage

### Browsing Products
1. Click "Shop" in the navigation menu
2. Filter by category or sort by price
3. Click "View Details" to see full product information
4. Select color, size, and quantity
5. Click "Add to Cart"

### Booking Custom Cloth
1. Click "Custom Order" in the navigation menu
2. Fill in all required information:
   - Personal details (name, phone, email)
   - Cloth specifications (type, description, color preferences)
   - Delivery date and time
3. Click "Send to WhatsApp"
4. Confirm the message in WhatsApp to send to GREENTHREAD

### Completing an Order
1. Browse and add products to cart
2. Click cart icon to review items
3. Click "Proceed to Checkout"
4. Fill in delivery information
5. Select payment method
6. Click "Send Order to WhatsApp"
7. Confirm in WhatsApp to send order
