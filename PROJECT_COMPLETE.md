## 🎓 How to Customize

### **Add New Products:**
Edit `src/data/products.js` and add to the products array.

### **Change Colors:**
Update the CSS variables in `src/App.css` and `src/index.css`:
```css
--primary-black: #1a1a1a;
--primary-gold: #d4af37;
--primary-white: #ffffff;
```

### **Update WhatsApp Number:**
Edit `src/utils/whatsappIntegration.js` and update:
```javascript
const WHATSAPP_NUMBER = "your_number_here";
```

### **Modify Product Categories:**
Edit `src/data/products.js` and update the categories array.
